# 💰 SmartMoney AI – Personal Finance Mentor

SmartMoney AI is an AI-powered personal finance assistant that helps users manage their income, expenses, and investments while providing intelligent financial insights.

---

## 🚀 Features

* 💰 Savings Calculation (Income - Expenses)
* 📈 SIP Recommendation (40% of savings)
* 🛟 Emergency Fund (6 months of expenses)
* 🧠 Money Health Score (out of 100)
* 📊 Portfolio Analysis:

  * XIRR (estimated return)
  * Portfolio Overlap (demo-based)
  * Expense Ratio Impact (~2% loss)
* 🤖 AI Suggestions (smart financial advice)

## ▶️ How to Run

### Simple Version:

1. Open `index.html` in browser
2. Enter values
3. Click "Analyze"

### Advanced Version:

1. Install dependencies:-
   pip install flask flask-cors

2. Run server:-
   python app.py

3. Open `index.html`
