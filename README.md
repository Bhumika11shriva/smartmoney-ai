# 💰 SmartMoney AI

SmartMoney AI is a simple AI-based personal finance mentor.

## 🚀 Features
- Calculates savings
- Suggests SIP investment
- Recommends emergency fund

## 🧑‍💻 Tech Stack
- HTML, CSS, JavaScript
- Python Flask

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run backend:
python app.py

3. Open index.html in browser

## 📊 Impact
Helps users improve savings and financial planning.
